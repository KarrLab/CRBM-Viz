More comprehensive and more predictive models have the potential to advance biology, bioengineering, and medicine. One of the most promising ways to build more comprehensive models is to combine multiple models of individual subsystems. For example, several researchers have are pioneering whole-cell models by combining multiple submodels that representing individual subcellular processes such as metabolism, transcription, and translation.

Building models by composition will likely require the collaborative effort of many computational and experimental biologists. Despite extensive efforts to develop standards such as COMBINE/OMEX , SBML , and SED-ML and repositories such as BioModels, it remains difficult to combine models. One challenge to composing models is the diverse array of incompatible modeling formats and simulation tools.

This application aims to help researchers combine models into more comprehensive models by providing researhcers a simple web interface for reusing any model. This is achieved through a collection of Docker images that abstract the details of multiple modeling frameworks, simulation algorithms, model formats, and simulation tools.

This application provides basic capabilities for executing simulations and visualizing their results. In addition to these features, the full-featured BioSimulations web application enables authors to publish models, simulations, and visualizations and provides users a central place to find and reuse published models, simulations, and visualizations. This application is ideal for investigators who wish to execute simulations privately during the course of their research. The full-featured application is ideal for investigators who want to publish modeling studies or reuse published studies.

This application can simulate logical, Flux Balance Analysis (FBA), continous kinetic (ordinary differential equations (ODE) and differential-algebraic equations (DAE)), and discrete kinetic (e.g., Stochastic Simulation Algorithms (SSA)) models that are described in using the BioNetGen Language (BNGL) or the Systems Biology Markup Language (SBML) .

This application uses the BioSimulators collection of simulations tools to simulate models. Through BioSimulators, BioSimulations supports a numerous algorithms for simulating logical, FBA, and kinetic models. BioSimulators builds upon Docker and BioContainers . Please see BioSimulators for more information about supported modeling frameworks, simulation algorithms, model formats, and simulation software tools.

This application is implemented using several open-source tools and cloud platforms.

The containerized simulation software tools are available from BioSimulators .

The source code for this application is available from GitHub .
